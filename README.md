# AI Todo

A personal todo app powered by GPT-4o mini. Add and manage tasks with natural language commands, and import ChatGPT conversations to auto-extract tasks, deadlines, and priorities.

## Setup

1. Clone the repo
2. `npm install`
3. `npm run dev`

## Deploy to Vercel

Push to GitHub, connect repo in Vercel — no environment variables needed (API key is stored in localStorage).
